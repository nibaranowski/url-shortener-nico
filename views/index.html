FreeCodeCamp API Basejump: URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://url-shortener-nico.glitch.me/new/https://www.google.com
 https://url-shortener-nico.glitch.me/new/http://foo.com:80

Example creation output

{ "originalUrl":"http://www.google.com, "shortenUrl":"https://url-shortener-nico.glitch.me/8653" }

Usage:

https://url-shortener-nico.glitch.me/8653

Example creation output

https://www.google.com/
